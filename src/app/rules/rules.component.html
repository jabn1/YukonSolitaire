How to play Yukon

The objective of the game is to move all the cards from the tableau to the foundation.

Card movement is done by dragging and dropping cards from on pile to the desired pile.

Aces rank lowest and Kings rank highest.

Cards can be moved in the tableau from one pile to another when placed in descending order and alternating in color. You can move multiple cards at a time between tableau piles. After removing all face up cards from a tableau pile with face-down cards, the top card will be revealed. An empty tableau pile can only be filled by a King.

Only one card at a time can be moved from the tableau to the foundation. Cards are placed in in the foundation in ascending order, starting with the Aces and can only be placed in the pile with the corresponding suit of the card. Once a card is placed in the foundation it cannot be removed.

Game options:

New game: this option will create a new game with a new deck of shuffled cards and your current game will be lost.

Restart game: this option will position the cards as they were initially dealt in the tableau at the start of the game and your current progress will be lost. This will reset the move count and timer to zero.

Undo: this option will undo the last move you made.

Redo: this option will revert the last undo action you made.

Pause/Resume: this option will pause the game if it is running; pausing the game causes the timer to stop while the game is paused, and the foundation and tableau will be hidden.
If the game is paused this option will resume the game; the cards will become visible and the timer will continue. While the game is paused the undo and redo buttons will have no effect.

Information:

Moves: this shows a counter that displays how many moves you have made in your current game. The first number is the moves in your current path and the second number is the total number of moves.
Note: using the undo and redo buttons will count as a move in the total number of moves; but using undo and redo will subtract and add moves to the number of moves of your current path.

Time: this displays a timer showing the total elapsed time of your current game.
